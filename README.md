##COMO RODAR O CÓDIGO

Criei um ambiente virtual pra rodarmos o código e instalei o pygame nele.

Para rodar, use na pasta do repositório

source .venv/bin/activate
python main.py

para sair do ambiente virtual, basta rodar

deactivate
